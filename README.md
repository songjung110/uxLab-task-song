## Description

### js-class-Chart 폴더

JS Class 문법을 사용한 Amcharts 3 / 4 / 5 버전 사용성 개발  
프론트 실무에서 사용할 수 있도록 나 (songjung110)이 정해둔 양식으로 정리해 둘 예정임.

> JS Class 문법을 사용한 Amcharts 3 / 4 / 5 버전 사용성 개발  
> 프론트 실무에서 사용할 수 있도록
>
> 1. 차트 ( 시각적 요소 )
> 2. JS Class 코드블럭
> 3. 실제 사용 예시 코드 블럭
>
> 해당 양식으로 정리 해 둘 예정 (현재 정리 안되어있음)

데이터 시각화용 실무 차트 테스트 페이지이지만, 사용하기 좋은 기능을 구현 할 경우 해당 페이지에 정리 해 둘 예정
